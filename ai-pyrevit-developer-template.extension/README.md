# pyRevit Extension Bundle

This is the folder pyRevit should load as an extension.

## Add To pyRevit
In Revit: `pyRevit` tab → `Settings` → `Extensions` → `Add`, then select this folder:

- `ai-pyrevit-developer-template.extension`

## What’s Inside
- `HelloWorld.tab/` - a single “Hello World” button to validate that pyRevit loads the extension.
