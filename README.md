# ai-pyrevit-developer-template

Minimal pyRevit extension template with a single placeholder command.

## Structure
- `Template.tab/Sample.panel/Placeholder.pushbutton/script.py`

## Use
1. Rename `Template.tab` and `Sample.panel` to your desired tab/panel names.
2. Replace `script.py` contents with your command.
